# Security Policy

## Supported Versions
Commerza is a proprietary project. Security fixes are applied to the current development version only.

## Reporting a Vulnerability
If you discover a security issue, please report it privately.

Contact:
- Email: syedahmershahofficial@gmail.com
- LinkedIn: https://www.linkedin.com/in/syedahmershah
- GitHub: https://github.com/ahmershahdev

Please include:
- A clear description of the issue
- Steps to reproduce
- Impact assessment
- Any relevant logs or screenshots

## Responsible Disclosure
Please do not disclose vulnerabilities publicly until you have received confirmation and a fix timeline.
