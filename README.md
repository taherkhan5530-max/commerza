# Commerza

Commerza is a full stack ecommerce project with a premium, conversion-focused storefront and a streamlined admin experience. It is built for fast merchandising, product discovery, and order management.

## Tech Stack
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- jQuery
- AJAX
- SEO-focused markup
- Laravel (backend)
- MySQL (database)

## Key Features
- Responsive storefront with category filtering, wishlist, compare, and product detail pages.
- Cart and checkout flow with order tracking.
- Newsletter modal and user account features.
- Admin panel for products, sections, website content, and order management.
- Local storage sync for fast prototyping with backend-ready structure.

## Getting Started
This repository includes front-end assets and admin UI. The backend is designed for Laravel + MySQL.

1. Clone the repository.
2. Serve the front-end pages with any static server for UI testing.
3. Wire up backend routes and controllers in Laravel as needed.

## Project Structure (High Level)
- Frontend pages: /index.html, /shop-category-a.html, /shop-category-b.html, /products.html
- Frontend assets: /frontend/assets
- Admin panel: /admin/frontend
- Data: /products.json
